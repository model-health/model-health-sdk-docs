      @modelhealth/modelhealth

      Model Health SDK Client

      TypeScript/JavaScript client for the Model Health biomechanics SDK. Provides a clean, typed API over the WASM bindings.

      Example: Basic usage

      import { ModelHealthService } from '@modelhealth/modelhealth';

      const client = new ModelHealthService({ apiKey: "your-api-key-here" });
      await client.init();

      const sessions = await client.sessionList();
