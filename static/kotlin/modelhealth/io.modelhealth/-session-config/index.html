SessionConfig

data class SessionConfig(val framerate: SessionFramerate = SessionFramerate.Fps120, val opensimModel: SessionOpenSimModel = SessionOpenSimModel.LaiUhlrich2022Shoulder, val scalingSetup: SessionScalingSetup = SessionScalingSetup.UprightStandingPose, val coreEngine: SessionCoreEngine = SessionCoreEngine.V0_3, val filterFrequency: FilterFrequency = FilterFrequency.Default, val dataSharing: SessionDataSharing = SessionDataSharing.ShareProcessedDataAndIdentifiedVideos)

Settings applied to a session before calibration and recording.

All fields have sensible defaults. Use SessionConfig.default or the no-argument constructor for a fully default configuration, then override only what you need:

val config = SessionConfig(framerate = SessionFramerate.Fps60, dataSharing = SessionDataSharing.ShareNoData)
