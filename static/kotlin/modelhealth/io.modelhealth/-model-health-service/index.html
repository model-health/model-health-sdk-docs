ModelHealthService

class ModelHealthService : Closeable

Main entry point for the Model Health Android SDK.

Create an instance with a valid API key. All API methods are suspend functions and must be called from a coroutine context. They dispatch blocking work to Dispatchers.IO.

val service = ModelHealthService("mh_your_api_key")
try {
    val sessions = service.sessionList()
} finally {
    service.close()
}
Content copied to clipboard

ModelHealthService implements Closeable so it can be used with Kotlin's use extension.

Thread safety: do not call methods on the same instance from multiple coroutines simultaneously. Create separate instances or serialize access externally if concurrent calls are required.
