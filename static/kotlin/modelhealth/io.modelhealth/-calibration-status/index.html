CalibrationStatus

@Serializable
sealed class CalibrationStatus

Status update from a camera or subject calibration operation.

Delivered to the CalibrationStatusCallback passed to ModelHealthService.calibrateCamera or ModelHealthService.calibrateSubject.

service.calibrateCamera(sessionId, checkerboard) { status ->
    when (status) {
        CalibrationStatus.Recording          -> showRecording()
        is CalibrationStatus.Uploading       -> showProgress(status.uploaded, status.total)
        is CalibrationStatus.Processing      -> showProcessing(status.percent)
        CalibrationStatus.Done               -> showDone()
    }
}
