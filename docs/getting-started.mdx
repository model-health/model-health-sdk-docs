## What you need to get started

You'll just need the following:

- **A laptop or tablet** of any type to run the web app at [app.modelhealth.io](https://app.modelhealth.io).
- **At least 2 iOS devices** — we support all iPhones and iPads released in 2018 or later. See the [full list of supported devices](#supported-ios-devices-for-recording) below.
- **Tripods and phone holders** for each iOS device — we've tested these specific [tripods](https://www.amazon.com/AmazonBasics-Lightweight-Camera-Mount-Tripod/dp/B00XI87KV8/) and [phone holders](https://www.amazon.com/Vastar-Universal-Smartphone-Horizontal-Adjustable/dp/B01L3B5PBI/) with excellent results.
- **Checkerboard** — print [this checkerboard](https://cdn.prod.website-files.com/62468717bed6b421c89bbf36/6258dd3518244b061c1b02f5_Checkerboard_4x5_35mm.pdf) on A4 paper and mount it to a flat surface. Each square must be exactly 35mm — print at actual size with no scaling.
- **A stable internet connection** — required for both the iOS app and web application to function properly.

<img src="/img/docs/getting-started/image.png" alt="Checkerboard setup" style={{width: '280px', marginLeft: '2rem', marginBottom: '0.5rem'}} />

:::warning
Print at **actual size (100% scale)** with no scaling and no "Fit to Page". The default square is exactly **35mm**.
You can also edit the square size to your printed checkerboard dimensions in the app settings — this is critical for accurate measurements.
:::

## Web App

The web app is where you review results, run advanced analyses, and export reports. Visit [app.modelhealth.io](https://app.modelhealth.io) to create an account. You can sign up for a 30-day free trial. Visit [modelhealth.io](https://www.modelhealth.io/) to learn more about subscription plans.

<details>
<summary><strong>Supported Browsers and Devices</strong></summary>

The web app works on all modern browsers. We recommend **Google Chrome** for the best experience.

| Browser | Supported | Laptop (Recommended for data analysis) | Tablet (Recommended for recording workflow) | Phones
|---|---|---|---|---|
| Google Chrome | ✅ Recommended |✅|✅|✅|
| Mozilla Firefox | ✅ |✅|✅|✅|
| Apple Safari | ✅ |✅|✅|✅|
| Microsoft Edge | ✅ |✅| | |

The web app works on laptops, tablets, and phones. We recommend using a **phone or tablet** for the most convenient on-site session review — a laptop is best for in-depth analysis and report exports.

</details>
