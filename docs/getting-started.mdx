## Requirement list

That's really all you need:

- **A laptop or tablet** — any type works, just open the web app at app.modelhealth.io.
- **At least 2 iOS devices** — any model released in 2018 or later. See the [full list of supported devices](#supported-ios-devices-for-recording) below.
- **Tripods and phone holders** — one per device. We've tested these specific [tripods](https://www.amazon.com/AmazonBasics-Lightweight-Camera-Mount-Tripod/dp/B00XI87KV8/) and [phone holders](https://www.amazon.com/Vastar-Universal-Smartphone-Horizontal-Adjustable/dp/B01L3B5PBI/) with excellent results.
- **Checkerboard** — print [this checkerboard](/img/docs/getting-started/Checkerboard_4x5_35mm.pdf) on A4 paper and mount it to a flat surface.

:::warning Measure your checkerboard squares
Print at **actual size (100% scale)** with no scaling and no "Fit to Page". The default square is exactly **35mm**.

In case your printed checkerboard does not have exactly 35mm squares, **update your checkerboard dimensions in the app Settings** — this is critical for accurate measurements.
<ZoomableImage src="/img/docs/getting-started/update-checkerboard-default.png" alt="Checkerboard setup" style={{width: '280px', marginLeft: '2rem', marginBottom: '0.5rem'}} />

:::

- **A stable internet connection** —  needed for both the app and web platform. Devices don't need to be on the same WiFi / mobile connection. Recording devices supported asynchronous video upload but require a minimal internet connection to communicate with the web app properly.

## Web App

The web app is where you review results, run advanced analyses, and export reports. Visit [app.modelhealth.io](https://app.modelhealth.io) to create an account. You can sign up for a 30-day free trial. Visit [modelhealth.io](https://www.modelhealth.io/) to learn more about subscription plans.

<details>
<summary><strong>Supported Browsers and Devices</strong></summary>

The web app works on all modern browsers. We recommend **Google Chrome** for the best experience.

| Browser | Supported | Laptop (Recommended for data analysis) | Tablet (Recommended for recording workflow) | Phones
|---|---|---|---|---|
| Google Chrome | ✅ Recommended |✅|✅|✅|
| Mozilla Firefox | ✅ |✅|✅|✅|
| Apple Safari | ✅ |✅|✅|✅|
| Microsoft Edge | ✅ |✅| | |

The web app works on laptops, tablets, and phones. We recommend using a **phone or tablet** for the most convenient on-site session review — a laptop is best for in-depth analysis and report exports.

</details>
