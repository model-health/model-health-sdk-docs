## System Requirements

You'll need the following:

- At least 2 iOS devices — we support iPhones, iPads and iPods released in 2018 or later.
- Tripods and phone holders — we've tested these specific [tripods](https://www.amazon.com/AmazonBasics-Lightweight-Camera-Mount-Tripod/dp/B00XI87KV8/) and [phone holders](https://www.amazon.com/Vastar-Universal-Smartphone-Horizontal-Adjustable/dp/B01L3B5PBI/) with excellent results.
- Printed checkerboard mounted to a flat surface — we recommend using [this checkerboard](https://cdn.prod.website-files.com/62468717bed6b421c89bbf36/6258dd3518244b061c1b02f5_Checkerboard_4x5_35mm.pdf). Mount it on a flat surface like a [plexiglass sheet](https://www.amazon.com/SimbaLux-Plexiglass-Transparent-Protective-Projects/dp/B078NJ1Q9W/), but avoid covering it with any reflective material such as lamination.

:::warning
Print the checkerboard at **actual size (100% scale)** with no scaling. The squares must be exactly **35mm wide** — this is crucial for accurate dimensions. Disable any "Fit to Page" option in your print settings.
:::

## Web App

Visit [app.modelhealth.io](https://app.modelhealth.io) to create an account. You can sign up for a 30-day free trial. Visit [modelhealth.io](https://www.modelhealth.io/) to learn more about subscription plans.

## Supported Browsers

Our web application is optimized for the following browsers:

- **Google Chrome** (recommended)
- **Mozilla Firefox**
- **Apple Safari**
- **Microsoft Edge**

We recommend running the web application from a laptop — it is not yet optimized for tablets and smartphones.

## Internet Connection

A stable internet connection is required for the iOS app and web application to function properly.
